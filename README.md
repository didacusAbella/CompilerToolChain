# CompilerToolChain

This repo contains the exercises solved during the  **Compiler Course** at University of Salerno. There are three softwares:

1. An handmade Lexer Analyzer. _(package: it.didacusabella.compilertoolchain.lexer)_;
2. A Lexer created with [JFlex](https://www.jflex.de/) _(package: it.didacusabella.compilertoolchain.flexlexer)_;
3. An handmande recursive descendant Top-Down Parser _(package: it.didacusabella.compilertoolchain.parser)_;


# Build Instructions

## Requirements
1. [Java](https://www.java.com/it/about/whatis_java.jsp) installed;
2. [Maven](https://maven.apache.org/) installed;

## Steps
1. `git clone https://github.com/didacusabella/CompilerToolChain`;
2. `mvn install`
